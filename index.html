Lewis Design System

A small set of CSS design tokens — colors, typography, spacing, radii — shared across Justin's projects. Drop in one stylesheet, and the same editorial visual identity carries through every project that links it.

Typography

Two-family system. Cormorant Garamond for display (h1 / h2 / large display numerals). Inter for body and UI. Mono falls back to the system stack.
